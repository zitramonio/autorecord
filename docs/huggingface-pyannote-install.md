https://huggingface.co/login

## Шаг 2. Примите условия доступа к модели

Откройте страницу модели:
https://huggingface.co/pyannote/speaker-diarization-community-1

На странице модели заполните данные вашей компании, Use case и нажмите кнопку согласия с условиями доступа:

Откройте страницу access tokens:
https://huggingface.co/settings/tokens

Выберите создание нового token с правами Read:

C:\Projects\autorecord\.worktrees\mvp\artifacts\hf-dialog-preview\Инструкция для скачивания с hugginface\hf7.png

После создания Hugging Face показывает token один раз. Скопируйте его сразу:
C:\Projects\autorecord\.worktrees\mvp\artifacts\hf-dialog-preview\Инструкция для скачивания с hugginface\hf8.png

## Шаг 4. Вставьте token в приложение

Вставьте token в поле внизу окна и нажмите `Скачать`.

Token используется только для скачивания этой модели и не сохраняется приложением.
